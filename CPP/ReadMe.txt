REQUIREMENTS
============
- This code is developed on Mac, and tested on VS2013 and g++,
- It can support Windows, Mac, Linux platforms
- c++ is based on C++ 11 standard


USAGE
=====
Linux:
    step 1:
        g++ TrainSolution.cpp
    step 2:
        ./a.out
Windows:
    step 1:
        Add the input.txt, TrainSolution.h, TrainSolution.cpp to your project
    step 2:
        run the solution
Mac:
    step 1:
        Add the input.txt, TrainSolution.h, TrainSolution.cpp to your project
    step 2:
        run the solution
